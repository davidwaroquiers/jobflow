API
===

The main interface for constructing workflows is the :obj:`Activity` object.

:obj:`Activity` objects contain :obj:`Job` and other :obj:`Activity` objects to execute.
Activities also contain useful functions for modifying and visualising the connectivity
of jobs within them.
